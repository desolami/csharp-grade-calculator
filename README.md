# C# Grade Calculator

This project is a beginner-friendly C# console application that calculates a student's grade based on the score entered by the user.

The program validates user input and assigns a grade using conditional statements.

## Features

- Accepts user score input
- Validates input using TryParse
- Assigns grades using if/else conditions
- Allows users to check another score

## Concepts Practiced

- Variables and data types
- User input with Console.ReadLine()
- Conditional statements (if / else)
- Input validation with TryParse
- While loops

## How to Run

1. Clone the repository
2. Open in Visual Studio or VS Code
3. Run the program
